📝 添加新项目指南

步骤 1: 创建项目文件夹

在 website/ 目录下创建新文件夹，如 my-project/

步骤 2: 创建 index.html

在新文件夹中创建 index.html 文件

步骤 3: 更新导航页

在 website/index.html 中添加新的项目卡片：

<a href="my-project/index.html" class="project-card">
    <div class="card-icon">📦</div>
    <h3>项目名称</h3>
    <p>项目描述</p>
    <span class="card-tag">标签</span>
</a>

步骤 4: 推送发布

git add .
git commit -m "Add: 新项目名称"
git push
